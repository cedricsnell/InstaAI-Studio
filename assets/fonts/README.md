# Font Assets

Place custom font files here (`.ttf`, `.otf`)

## Default Fonts

InstaAI Studio uses system fonts by default:
- **Arial-Bold**: Clean, professional
- **Impact**: Bold, attention-grabbing
- **Comic-Sans-MS**: Casual (use sparingly!)
- **Courier**: Monospace, technical look

## Adding Custom Fonts

1. Download `.ttf` or `.otf` font files
2. Place them in this directory
3. Reference by filename (without extension) in commands

## Usage

```bash
# Using default font
instaai create video.mp4 "Add text 'Hello' at the center"

# Using custom font (if you have custom.ttf in this folder)
# You'll need to modify the code to support custom font loading
```

## Recommended Fonts for Instagram

- **Montserrat**: Modern, clean
- **Roboto**: Professional, versatile
- **Bebas Neue**: Bold headers
- **Oswald**: Strong, impactful
- **Raleway**: Elegant, minimal
