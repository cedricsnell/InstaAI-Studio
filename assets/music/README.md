# Music Assets

Place your background music files here (`.mp3`, `.wav`, `.m4a`, etc.)

## Recommendations

- Use royalty-free music to avoid copyright issues
- Keep files under 10MB for faster processing
- Common genres for Instagram:
  - Upbeat/Energetic for product launches
  - Chill/Lofi for tutorials
  - Trending audio for viral content

## Free Music Resources

- [YouTube Audio Library](https://www.youtube.com/audiolibrary)
- [Free Music Archive](https://freemusicarchive.org/)
- [Incompetech](https://incompetech.com/)
- [Bensound](https://www.bensound.com/)
- [Purple Planet](https://www.purple-planet.com/)

## Usage

Once you add music files here, you can reference them in commands:

```bash
instaai create video.mp4 "Add upbeat.mp3 as background music at 50% volume"
```

Or let AI choose:

```bash
instaai create video.mp4 "Add energetic background music"
```
